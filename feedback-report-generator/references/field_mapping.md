# Excel 字段映射说明

## 必填字段

| 脚本字段 | 常见 Excel 列名（任一匹配即可） | 说明 |
|---------|-------------------------------|------|
| `cat` | 品类、category、产品品类 | 用于按品类分组，每个品类独立分析 |
| `eval_type` | 评价、满意度、用户评价 | 满意度，必须是：`非常不满意` / `一般` / `满意` 之一 |
| `survey` | 问卷环节、环节、问卷阶段 | 用户反馈发生的流程环节，如：选机型、质检、下单、服务、估价 |
| `desc` | 描述、用户描述、反馈内容、description | 用户的文字反馈内容 |
| `tags` | 标签、问题标签 | 问题分类标签，多标签用 `|` 分隔，如：`缺少品牌|价格不满意` |

## 可选字段

| 脚本字段 | 常见 Excel 列名 | 说明 |
|---------|----------------|------|
| `time` | 提交时间、时间、日期 | 反馈提交时间，显示在明细表里 |
| `img` | 图片、图片url、image | 图片 URL（必须以 http 开头），多图用 `|` 分隔 |
| `est_price` | 预估价、estimated_price | 预估回收价格 |
| `check_price` | 质检价、check_price | 质检后实际价格 |
| `score` | 评分、score、星级 | 评分数字（1-5），显示为星级 |

## 标签→分类 映射规则

脚本会根据标签自动将每条反馈归入以下分类（一条可同时属于多个分类）：

| 标签 | 归入分类 | 分类名称 |
|-----|---------|---------|
| 缺少品牌、缺少型号、缺少商品、缺少品类、缺少实拍图 | A | 品类/品牌/型号缺失 |
| 价格不满意 | B | 回收价格不满意 |
| 质检结果存疑 | C | 质检结果存疑 |
| 催促验机/物流、服务质量、联系不上工程师 | D | 物流/服务问题 |
| 密码账号相关、不清楚如何操作 | E | 操作/功能问题 |
| 其他（未匹配任何分类） | MISC | 其他反馈 |

**扩展标签映射**：在 `gen_report.py` 的 `TAG_CLASS` 字典中添加新标签即可。

## 注意事项

1. **多标签行**：一行数据如果同时有 `缺少品牌|价格不满意`，会同时出现在 A 类和 B 类的明细里，两个分类的条数统计各自计入。这是正确行为。
2. **列名匹配**：不区分大小写，支持前后空格。如果列名完全不匹配，对应字段为空（不影响生成，仅该列显示空）。
3. **评价字段**：如果值不是 `非常不满意/一般/满意` 三选一，差评率统计会偏低，建议在 Excel 里预处理。

## 快速测试

```bash
python3 gen_report.py your_data.xlsx
# 输出：your_data_分析报告.html

python3 gen_report.py your_data.xlsx output/report.html
# 指定输出路径
```
